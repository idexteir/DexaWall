# Dexawall 

**The Ultimate Lightweight Wallpaper Manager for Windows**

Dexawall is a modern, portable, and high-performance wallpaper manager designed to breathe life into your desktop. Built with .NET 8 and WPF, it offers a seamless experience for managing local libraries and discovering stunning 4K/8K wallpapers online via Wallhaven.
